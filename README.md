# Overview

This is a sample hello world software written in rust.

This software would print a "Hello World" text in the console and then exit.

This is a demostration to show that the fundamentals of the rust compiler are correctly set up in the host system and does indeed run - take in code and oroduce an output to console.

See link to demostration of working software below:

[Software Demo Video](http://youtube.link.goes.here)

# Development Environment

Built using VS Code running C++ compiler and build tools. Rust language toolschain and cargo package manager

Software written using Rust lnaguage. Rust is a great language due to its in-built memory managment that comes without a gabbage collector. This prevents buffer overflows and other memory issues common in most other languages.
